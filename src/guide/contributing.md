## 依赖环境
请先下载并安装 [Node.js](https://nodejs.org/)。
- [Node.js v16.19.0+](https://nodejs.org/)

## 安装
```bash
# clone 本项目
git clone https://github.com/choushunn/intelligent-perception-doc
# 进入项目目录
cd intelligent-perception-doc
# 安装项目依赖
npm install
```

## 启动

```
# 运行项目
npm run docs:dev
```
